# Руководство по развертыванию AI-инфраструктуры (Disaster Recovery)

Данный документ описывает пошаговый процесс полного восстановления всех ИИ-агентов, их логики, Python-скриптов и настроек на новом или отформатированном компьютере.

**ВАЖНО:** Для восстановления системы у вас должен быть доступ к вашему аккаунту на **GitHub.com** и к рабочей директории с исходными Excel-файлами (Workplace).

---

## 1. Базовая установка программ (Software Stack)

На "чистом" компьютере необходимо установить 4 базовые программы. При установке везде используйте параметры по умолчанию.

1. **Python 3.12+**
   - Скачайте с официального сайта python.org.
   - **КРИТИЧНО:** На первом экране установщика обязательно поставьте галочку **"Add Python to PATH"**, иначе скрипты не будут запускаться.
2. **Git для Windows**
   - Скачайте с gitforwindows.org.
   - Оставьте все настройки по умолчанию (включая Git Credential Manager и main branch).
3. **Docker Desktop**
   - Скачайте с официального сайта Docker.
   - После установки запустите Docker Desktop и примите лицензионное соглашение. Программа должна работать в фоновом режиме.
4. **Google Antigravity / Gemini Agent Framework**
   - Установите локальную ИИ-среду.

---

## 2. Восстановление "Мозгов" из GitHub (Clone)

Вся логика ваших бизнес-процессов хранится в облаке. Чтобы скачать её:

1. Откройте папку "Документы" (C:\Users\ВашПользователь\Documents).
2. Кликните правой кнопкой мыши по пустому месту в папке и выберите **"Open Git Bash here"**.
3. Введите команду:
   `ash
   git clone https://github.com/DG-667/AI_Brains.git
   `
4. В этот момент появится всплывающее окно авторизации. Войдите через браузер.
5. На вашем компьютере появится папка AI_Brains со всеми скриптами и конфигурациями агентов.

---

## 3. Импорт Агентов (Agent Setup)

В папке AI_Brains/agents/ лежат файлы .json. Это "души" агентов: их системные промпты, правила и роли.

1. Запустите ИИ-агента (Antigravity).
2. Напишите ему первый запрос: 
   > *"Я восстанавливаю систему на новом ПК. В папке C:\Users\...\Documents\AI_Brains\agents\ лежат файлы конфигурации агентов. Считай их и восстанови (define_subagent) всю команду агентов."*
3. Агент прочитает эти JSON-файлы и мгновенно воссоздаст всех ваших виртуальных сотрудников.

---

## 4. Развертывание Docker-контейнеров

1. Агент (которому вы дадите задачу) сам зайдет в папку AI_Brains/docker_agents/.
2. Он прочитает файл docker-compose.yml и запустит все необходимые сервисы. Вам ничего настраивать не нужно.

---

## 5. Восстановление Данных

1. Перенесите вашу папку Workplace (где лежат сырые Excel-файлы GfK, еженедельные отчеты и т.д.) на новый компьютер по старому пути:
   C:\Users\...\Documents\Workplace\Data\...
2. Поздравляю! Инфраструктура полностью восстановлена. Система готова к бою.
