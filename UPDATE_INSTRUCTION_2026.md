# Обновление Команды Агентов (Июнь 2026)

В рамках последних доработок наша команда ИИ-агентов была существенно модернизирована. Ниже приведено описание ключевых изменений, новых правил и настроек.

## 1. Новый алгоритм работы Market Analyst
Агент `market_analyst` был переведен с простой экстраполяции тренда на **драйвер-ориентированный прогноз**. 
- **Источники данных:** Агент теперь жестко привязан к парсингу реальных исторических объемов из GfK (Excel).
- **Методология:** Включена строгая декомпозиция продаж (тренд + сезонность + остаток).
- **Лаги и эластичность:** Агент оценивает макроэкономические драйверы (курс KZT/USD, реальные доходы, ставки по рассрочке) с применением временных лагов (например, лаг 2 месяца для курса валют).
- **Структурные разрывы:** Добавлено обязательное вычленение шоковых периодов (ре-экспорт в РФ 2022-2023 гг., циклы обновления парка после пандемии и окончания поддержки Windows 10).
- **Результат:** Формируются 3 независимых сценария (Базовый, Оптимистичный, Пессимистичный) с помесячной детализацией, без использования жестких заглушек (hardcaps).

## 2. Глобальная изоляция в Docker (Security Rule)
Для защиты локальной среды (хост-машины Windows) от мусора и конфликтующих библиотек введено глобальное системное правило в `.gemini/config/AGENTS.md`:
```markdown
# DOCKER ISOLATION RULE  
All agents MUST execute their Python scripts and arbitrary code INSIDE a Docker container. Do NOT run code directly on the host Windows machine.
```
Теперь, когда агенты пишут Python-код (например, скрипты на базе `pandas` и `statsmodels`), они обязаны собирать под этот скрипт временный Docker-контейнер и выполнять вычисления внутри него.

## 3. Подключение нативных MCP-серверов
Чтобы избавиться от необходимости писать Python-код для каждой рутинной задачи, мы интегрировали мощные MCP (Model Context Protocol) серверы прямо в IDE:
- **`github-mcp-server`**: Позволяет агентам напрямую делать коммиты, создавать ветки, управлять Issue и читать код в репозиториях GitHub. Настроен через Docker и `Personal Access Token` (repo scope).
- **`@negokaz/excel-mcp-server`**: Запускается через `npx` (Node.js). Дает агентам нативные инструменты для прямого чтения и записи листов Excel, форматов и формул без запуска внешних скриптов (идеально для трекера промо-активностей).
- **`sequential-thinking`**: Сервер для продвинутого логического планирования.
